# ContractorFlow CRM

A modern React + Tailwind contractor CRM dashboard for small remodeling, deck, roofing, and painting businesses.

## Features

- Responsive SaaS-style dashboard
- Dark sidebar and sticky top navigation
- Dashboard metrics
- Drag-and-drop lead pipeline
- Mock contractor leads
- Reusable React components
- Tailwind CSS styling

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in your terminal.
