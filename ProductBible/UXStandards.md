# UX Standards

- Use modals for focused create, edit, confirm, upload, send, and payment actions when that pattern already exists.
- Show toast feedback for successful actions, restores, archives, deletes, and other key state changes.
- Keep navigation predictable across sidebar, topbar, and mobile flows.
- Follow the archive pattern: archive hides active records, restore brings them back, delete is permanent.
- All visible labels, buttons, helper text, and empty states must exist in both English and Spanish through translation keys.
- Never ship visible UI actions that do nothing.
