# ADR-003: Bilingual, Mobile-First

Status: Accepted

Context
- Target users are small Hispanic contractors who frequently work from phones.
- Accessibility, localization, and mobile usability are critical for adoption.

Decision
- Design and build the app mobile-first and support English and Spanish (both languages maintained together).

Consequences
- UI and layouts prioritized for small screens; translations must be kept in sync.
- Slightly higher initial design/QA effort, but better fit for target users and adoption.
