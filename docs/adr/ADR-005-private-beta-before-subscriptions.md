# ADR-005: Private Beta Before Subscriptions

Status: Accepted

Context
- Early product-market fit is unknown; billing infrastructure adds complexity and cost.
- We can learn more from a small group of real contractors before offering paid plans.

Decision
- Run a private beta with 1–3 contractors to validate value and flows before implementing Stripe/subscriptions.

Consequences
- Faster learning and lower upfront infrastructure cost.
- Delay in revenue capture; subscription work scoped after validated demand.
- Beta feedback will shape pricing and feature priorities.
